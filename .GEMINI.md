## 🚀 開発ルール

1. **クライアントサイド完結**: サーバーサイド処理は行わない (Next.js App Router + Static Export)。
2. **デザイン優先**: Tailwind CSS を使用し、Dark Mode & Glassmorphism を採用。
3. **パフォーマンス**: Canvas API を最適化し、ネットワーク遅延ゼロを目指す。
4. **言語**: コード内のコメント、コミットメッセージ、ドキュメントは日本語を使用（コード自体は英語）。
